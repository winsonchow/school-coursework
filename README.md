# School Coursework on Machine Learning Projects

Welcome to my GitHub repository where I've compiled some of my coursework projects related to machine learning. Each project focuses on different aspects of machine learning, ranging from natural language processing to image classification and regression modeling. Below, you'll find a brief overview of each project:

## 1. BERT-Based Topic Modelling

In this project, I explored the world of natural language processing by utilizing the powerful BERT (Bidirectional Encoder Representations from Transformers) model for topic modeling. I used a dataset containing textual data and employed BERT's language understanding capabilities to extract meaningful topics from the text. The project highlights my understanding of advanced NLP techniques and their practical applications.

## 2. Image Classification on CIFAR-10

The CIFAR-10 dataset is a widely used benchmark for image classification tasks. In this project, I delved into the field of computer vision by developing an image classification model using deep learning techniques. I trained and fine-tuned a convolutional neural network (CNN) on the CIFAR-10 dataset, achieving accuracy and demonstrating my skills in working with image data and neural networks.

## 3. Regression Modelling on Ames Housing

Real-world data often requires regression analysis to make predictions or understand relationships between variables. In this project, I tackled the Ames Housing dataset, a classic dataset for regression tasks. Through data preprocessing, feature engineering, and model selection, I built regression models to predict housing prices. This project showcases my proficiency in regression modeling and feature manipulation.

## How to Use This Repository

Each project is contained in its respective folder, along with the necessary code, datasets (if applicable), and a Jupyter Notebook detailing the project's steps, methodologies, and results. Feel free to explore each folder, review the code, and refer to the Jupyter Notebooks for comprehensive explanations.
